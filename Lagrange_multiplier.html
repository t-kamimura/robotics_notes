1. 停留値問題

ある多変数関数f(x1,x2,⋯ ,xn)f(x_1, x_2, \cdots, x_n)f(x1​,x2​,⋯,xn​)の停留値問題を考える．極小値(あるいは極大値)を考えることに似ているが，とにかく傾きがゼロになる点を求める(例えばf(x)=x3f(x)=x^3f(x)=x3におけるx=0x=0x=0は極大でも極小でもないが停留値である)．
nnn次元を考えるのは大変なので，2変数関数程度のものをイメージすると良い．

停留点においては，わずかにxix_ixi​の値を変化させたところで関数の返り値はほとんど変化しない．すなわち，関数fffの停留値からのズレをδf\delta fδfで表すと，
δf=∂f∂x1δx1+⋯+∂f∂xiδxi+⋯+∂f∂xnδxn=0\delta f= \frac{\partial f}{\partial x_1}\delta x_1 + \cdots + \frac{\partial f}{\partial x_i}\delta x_i + \cdots + \frac{\partial f}{\partial x_n}\delta x_n = 0δf=∂x1​∂f​δx1​+⋯+∂xi​∂f​δxi​+⋯+∂xn​∂f​δxn​=0
が成立する．このように点を（わずかに）変化させたときの変位を仮想変位と呼ぶ．
この式だけでは，例えばいい感じに∂f/∂xi\partial f/\partial x_i∂f/∂xi​と∂f/∂xj\partial f/\partial x_j∂f/∂xj​が打ち消し合った結果，δf=0\delta f=0δf=0となっても構わないのだが，停留値であるという条件下においては，いかなるδx\delta xδxについてもδf=0\delta f = 0δf=0が成立する（例えばボウルの底，あるいは鞍点のような点をイメージすると良い）．すなわち，
∂f∂xi=0\frac{\partial f}{\partial x_i} = 0∂xi​∂f​=0
がすべてのiiiについて成立する．これが停留値となるための条件である．

2. 1個の拘束条件を伴う停留値問題

上記では，すべての変数xix_ixi​が独立であると仮定していた．ここでは，変数が独立ではない場合を考えよう．すなわち，
g(x1,x2,⋯ ,xn)=0g(x_1, x_2, \cdots, x_n)=0g(x1​,x2​,⋯,xn​)=0
という拘束条件が存在するときのfffの停留値問題を考える．ここでは，拘束条件がx1,x2,⋯ ,xnx_1, x_2, \cdots, x_nx1​,x2​,⋯,xn​だけで表され，これらの微分にはよらない場合を考えよう．このような条件をホロノミック拘束と呼ぶ．

拘束条件が存在すると，もはや各変数は独立変数ではなく，仮想変位を考えるときにも制約が生まれてしまう．そうなると停留値問題は単純に
∂f∂xi=0\frac{\partial f}{\partial x_i} = 0∂xi​∂f​=0
が満たされればよいというわけにはいかなくなる．

2.1. 従属変数を削除する方法

ホロノミックな拘束条件を用いると，変数の数を減らして考えることができる．すなわち，ある変数xkx_kxk​（問題を解く上で都合の良いものを選べばよい）を選び，それを他のn−1n-1n−1個の独立変数で表すことができる．ここでは簡単のため，
xn=xn(x1,x2,…,xn−1)x_n = x_n(x_1, x_2, \dots, x_{n-1})xn​=xn​(x1​,x2​,…,xn−1​)
としてみよう．x1,x2,…,xn−1x_1, x_2, \dots, x_{n-1}x1​,x2​,…,xn−1​は完全に独立な変数となる．
このとき，
∂f∂xi=0\frac{\partial f}{\partial x_i} = 0∂xi​∂f​=0
がすべてのi=1,2,⋯ ,n−1i = 1, 2, \cdots, n-1i=1,2,⋯,n−1に対して満たされているのがfffの停留値である．

2.2. 未定乗数を用いる方法

上記の方法は理解しやすい．しかし，従属変数を恣意的に選択する必要がある上，場合によっては式の対称性を失ってしまうことにもなりかねない．また，拘束条件が複雑で，従属変数が簡単に書き下さえない場合も存在するだろう．このような場合に用いたいのが，以下に示すラグランジュの未定乗数法である．

まず，停留値においては拘束条件の有無に関わらず
δf=∂f∂x1δx1+⋯+∂f∂xiδxi+⋯+∂f∂xnδxn=0\delta f= \frac{\partial f}{\partial x_1}\delta x_1 + \cdots + \frac{\partial f}{\partial x_i}\delta x_i + \cdots + \frac{\partial f}{\partial x_n}\delta x_n = 0δf=∂x1​∂f​δx1​+⋯+∂xi​∂f​δxi​+⋯+∂xn​∂f​δxn​=0
が成立する．
さらに，拘束条件g=0g=0g=0から
δg=∂g∂x1δx1+⋯+∂g∂xiδxi+⋯+∂g∂xnδxn=0\delta g= \frac{\partial g}{\partial x_1}\delta x_1 + \cdots + \frac{\partial g}{\partial x_i}\delta x_i + \cdots + \frac{\partial g}{\partial x_n}\delta x_n = 0δg=∂x1​∂g​δx1​+⋯+∂xi​∂g​δxi​+⋯+∂xn​∂g​δxn​=0
が成立する．
ここで，未定定数と呼ばれるある定数λ\lambdaλを用いて，

∂f∂x1δx1+⋯+∂f∂xiδxi+⋯+∂f∂xnδxn+λ(∂g∂x1δx1+⋯+∂g∂xiδxi+⋯+∂g∂xnδxn)=0\frac{\partial f}{\partial x_1}\delta x_1 + \cdots + \frac{\partial f}{\partial x_i}\delta x_i + \cdots + \frac{\partial f}{\partial x_n}\delta x_n \\+ \lambda \left(\frac{\partial g}{\partial x_1}\delta x_1 + \cdots + \frac{\partial g}{\partial x_i}\delta x_i + \cdots + \frac{\partial g}{\partial x_n}\delta x_n \right)=0 ∂x1​∂f​δx1​+⋯+∂xi​∂f​δxi​+⋯+∂xn​∂f​δxn​+λ(∂x1​∂g​δx1​+⋯+∂xi​∂g​δxi​+⋯+∂xn​∂g​δxn​)=0

とする．0+λ×00+\lambda\times00+λ×0という形になっているので，当然=0=0=0が成立する．

（注）未定乗数が突然現れたが，動じてはいけない．ゼロにゼロを足すというこの謎の操作こそがラグランジュの未定乗数法の核をなすアイデアである．

式が長いので，シグマを用いてまとめると，
∑k=1n(∂f∂xk+λ∂g∂xk)δxk\sum_{k=1}^n \left(\frac{\partial f}{\partial x_k} + \lambda\frac{\partial g}{\partial x_k}\right)\delta x_kk=1∑n​(∂xk​∂f​+λ∂xk​∂g​)δxk​
となる．
ここで，中身の決まっていなかったλ\lambdaλの値を
∂f∂xn+λ∂g∂xn=0\frac{\partial f}{\partial x_n} + \lambda\frac{\partial g}{\partial x_n} = 0∂xn​∂f​+λ∂xn​∂g​=0
を満たすように決定してみよう．このとき，
∑k=1n−1(∂f∂xk+λ∂g∂xk)δxk=0\sum_{k=1}^{n-1} \left(\frac{\partial f}{\partial x_k} + \lambda\frac{\partial g}{\partial x_k}\right)\delta x_k = 0k=1∑n−1​(∂xk​∂f​+λ∂xk​∂g​)δxk​=0
が成立する．任意のδxk\delta x_kδxk​に対してこれが成立するためには，
∂f∂xk+λ∂g∂xk=0\frac{\partial f}{\partial x_k} + \lambda\frac{\partial g}{\partial x_k} = 0∂xk​∂f​+λ∂xk​∂g​=0
がk=1,2,⋯ ,n−1k = 1, 2, \cdots, n-1k=1,2,⋯,n−1について成立する必要がある．

以上をまとめると，i=1,2,⋯ ,ni = 1,2,\cdots, ni=1,2,⋯,nについて
∂f∂xi+λ∂g∂xi=0\frac{\partial f}{\partial x_i} + \lambda\frac{\partial g}{\partial x_i} = 0∂xi​∂f​+λ∂xi​∂g​=0
が成立するのが，拘束条件を満たした上での停留値であるということになる．
ここから，ホロノミックな拘束条件を含む停留値問題は，
fˉ=f+λg\bar{f} = f + \lambda gfˉ​=f+λg
においてx1,x2,⋯ ,xnx_1, x_2, \cdots, x_nx1​,x2​,⋯,xn​がすべて独立変数だとみなしたときの停留値問題に帰着する．

3. 未定定数の幾何的解釈

ここまでの説明は少し抽象的だったので，具体的なイメージを持っておこう．
拘束条件g(x1,⋯ ,xn)=0g(x_1,\cdots,x_n)=0g(x1​,⋯,xn​)=0は，nnn次元空間内のn−1n-1n−1次元の曲面（拘束面）を表す．
拘束条件の勾配∇g=[∂g/∂x1,⋯ ,∂g/∂xn]⊤\nabla g = [\partial g/\partial x_1, \cdots, \partial g/ \partial x_n]^\top∇g=[∂g/∂x1​,⋯,∂g/∂xn​]⊤を取ると，これは拘束面に対して常に垂直である．これは以下のようにして確かめられる．

拘束面上の2点xxxとx+εx+\varepsilonx+εを考えたとき（ε\varepsilonεは非常に小さいとする），xxxの周りのテイラー展開により
g(x+ε)=g(x)+ε⊤∇g(x)+O(∣∣ε∣∣2)g(x+\varepsilon)=g(x)+\varepsilon^\top \nabla g(x) + O(||\varepsilon||^2)g(x+ε)=g(x)+ε⊤∇g(x)+O(∣∣ε∣∣2)
を得る．xxxもx+εx+\varepsilonx+εも拘束面上にあることから，g(x)=g(x+ε)=0g(x)=g(x+\varepsilon)=0g(x)=g(x+ε)=0であり，2次以上の微小項が無視できるほど小さいとき，
ε⊤∇g(x)\varepsilon^\top \nabla g(x)ε⊤∇g(x)
が成立する．これは∇g(x)\nabla g(x)∇g(x)と，ε\varepsilonε（拘束面に平行）が垂直であることを意味している．

さて，もとの問題に戻ろう．f(x)f(x)f(x)が停留値を取るx∗x^*x∗を拘束面上で探す．
例えばf(x)f(x)f(x)という山に，g(x)=0g(x)=0g(x)=0という道があるというイメージを持とう．このときg(x)=0g(x)=0g(x)=0上を移動しながら，標高が最も高くなる地点x∗x^*x∗を探す．
x∗x^*x∗においては∇f(x∗)\nabla f(x^*)∇f(x∗)も拘束面に対して垂直でなければならない．なぜなら，仮に垂直でないとすると，拘束面に沿ってさらに大きいf(x)f(x)f(x)の値が取れるから．
したがって，∇f\nabla f∇fと∇g\nabla g∇gは（向きが逆の場合も含めて）平行なベクトルとなる．すなわち，
∇f+λ∇g=0\nabla f + \lambda \nabla g = 0∇f+λ∇g=0
が停留条件として書き直せることになる．ここでλ\lambdaλはラグランジュの未定定数であり，正の値でも負の値でもありえる．

ここで，ラグランジュ関数
L(x,λ)=f(x)+λg(x)L(x,\lambda) = f(x) + \lambda g(x)L(x,λ)=f(x)+λg(x)
を定義すると，停留条件は∇xL=0\nabla_x L = 0∇x​L=0と表される．また，∂L/∂λ=0\partial L/\partial \lambda = 0∂L/∂λ=0も成立する．

以上をまとめると，拘束条件g(x)=0g(x)=0g(x)=0のもとでf(x)f(x)f(x)の停留値をもとめる問題では，ラグランジュ関数L(x,λ)L(x,\lambda)L(x,λ)のxxxとλ\lambdaλの両方に対する停留点を求める問題に帰着する．
xxxがnnn次元のベクトルとすると，n+1n+1n+1本の方程式が得られるので，それらを解けばx∗x^*x∗とλ\lambdaλが求められる．

4. 複数の拘束条件を持つ停留値問題

拘束条件が

g1(x1,x2,⋯ ,xn)=0g2(x1,x2,⋯ ,xn)=0⋮gm(x1,x2,⋯ ,xn)=0g_1(x_1, x_2, \cdots, x_n) = 0\\ g_2(x_1, x_2, \cdots, x_n) = 0\\ \vdots\\ g_m(x_1, x_2, \cdots, x_n) = 0\\ g1​(x1​,x2​,⋯,xn​)=0g2​(x1​,x2​,⋯,xn​)=0⋮gm​(x1​,x2​,⋯,xn​)=0

で表されるときも同様に考えて，i=1,2,⋯ ,ni = 1,2,\cdots, ni=1,2,⋯,nについて
∂f∂xi+λ1∂g1∂xi+λ2∂g2∂xi⋯+λm∂gm∂xi=0\frac{\partial f}{\partial x_i} + \lambda_1\frac{\partial g_1}{\partial x_i} + \lambda_2\frac{\partial g_2}{\partial x_i} \cdots + \lambda_m\frac{\partial g_m}{\partial x_i} = 0∂xi​∂f​+λ1​∂xi​∂g1​​+λ2​∂xi​∂g2​​⋯+λm​∂xi​∂gm​​=0
が成立するのが，拘束条件を満たした上での停留値であるということになる．

5. 例題

イメージがつかない可能性があるので，例題を問いておこう．
x2+y2=1x^2 + y^2=1x2+y2=1の単位円上で，f(x)=x2+xy+y2f(x) = x^2 + xy + y^2f(x)=x2+xy+y2が取りうる値の極値を求めよう．

拘束条件は
g(x,y)=x2+y2−1=0g(x,y) = x^2 + y^2 - 1 = 0g(x,y)=x2+y2−1=0
と書ける．
未定乗数λ\lambdaλを用いて，

∂f∂x+λ∂g∂x=(2+2λ)x+y=0∂f∂y+λ∂g∂y=x+(2+2λ)y=0\frac{\partial f}{\partial x} + \lambda\frac{\partial g}{\partial x} = (2+2\lambda)x + y = 0\\ \frac{\partial f}{\partial y} + \lambda\frac{\partial g}{\partial y} = x + (2+2\lambda)y = 0 ∂x∂f​+λ∂x∂g​=(2+2λ)x+y=0∂y∂f​+λ∂y∂g​=x+(2+2λ)y=0

ここから，λ=−1/2,−3/2\lambda = -1/2, -3/2λ=−1/2,−3/2が求められる．
λ=−1/2\lambda = -1/2λ=−1/2に対して
x+y=0x + y = 0x+y=0
となり，x2+y2=1x^2 + y^2 = 1x2+y2=1と連立して，x=±2,y=∓2x=\pm \sqrt{2}, y = \mp \sqrt{2}x=±2​,y=∓2​を得る．このときfffは極小値−1-1−1を取る．
λ=−3/2\lambda = -3/2λ=−3/2に対して
−x+y=0-x + y =0−x+y=0
となり，x2+y2=1x^2 + y^2 = 1x2+y2=1と連立して，x=±2,y=±2x=\pm \sqrt{2}, y = \pm \sqrt{2}x=±2​,y=±2​を得る．このときfffは極大値333を取る．
