# Mapleで微分する

計算ソフトウェアMapleを使用すると，さまざまな解析計算が可能である．
ここではよく用いる微分演算についてまとめる．

## 1変数による微分

1変数による微分には`diff`関数を用いる．
```
> restart
> f := x -> x^2 + 3*x + 2;
> diff(f(x), x)
                            2x + 3
```

多変数関数に対して`diff`を施してもよい．
```
> restart
> f := (x, y) -> x^2 + 3*x + 6 + y^2 + 4*y:
> diff(f(x, y), x);
                            2x + 3
> diff(f(x, y), y);
                            2y + 4
```

## ヤコビ行列

まとめて偏微分したいときには`Jacobian`を用いる．
