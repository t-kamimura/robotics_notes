工学部に入ってはじめに学ぶ数学は，微分積分学と線形代数学の２つであることが多いと思います．
学部の線形代数学では特に，掃き出し法とか対角化のアルゴリズムを覚えて手計算を行えるようになることに尽力してしまい，結局何を学んだのかわからないままになっている方が多い印象があります．かくいう筆者もそのような学生の一人でした．
ここでは，細かい計算を抜きにして，最速で対角化の理論までたどり着き，１年生の線形代数学で何を学んだのかを整理してみようと思います．

本稿では，紙面の都合上，細かい用語の定義や，定理の証明は飛ばして自由に用いることにしますので，必要に応じて適宜線形代数の教科書を参照して下さい．

1. ベクトル空間

1.1. ベクトル空間とベクトルの定義

線形代数ではまずベクトルを定義します．ここでベクトルとは，高校生で習った「向きと大きさを持つ量」というイメージを超えて，かなり抽象的な定義を行います．

まず，ベクトル空間を定義します．
ベクトル空間とは，以下のような性質をもつものの集合です．

  1. u+v=u+v\bm{u}+\bm{v} = \bm{u}+\bm{v}u+v=u+v
  2. (u+v)+w=u+(v+w)(\bm{u}+\bm{v})+\bm{w} = \bm{u}+(\bm{v}+\bm{w})(u+v)+w=u+(v+w)
  3. v+0=v\bm{v}+\bm{0}=\bm{v}v+0=vとなる元0\bm{0}0が存在する
  4. v\bm{v}vに対してv+v′=0\bm{v} + \bm{v'}=\bm{0}v+v′=0となる元v′=−v\bm{v'}=-\bm{v}v′=−vが存在する
  5. c(u+v)=cu+cvc(\bm{u}+\bm{v}) = c\bm{u}+c\bm{v}c(u+v)=cu+cv
  6. (c1+c2)v=c1v+c2v(c_1+c_2)\bm{v} = c_1 \bm{v} + c_2\bm{v}(c1​+c2​)v=c1​v+c2​v
  7. (c1c2)v=c1(c2v)(c_1c_2)\bm{v} =c_1 (c_2\bm{v})(c1​c2​)v=c1​(c2​v)
  8. 1v=v1\bm{v} = \bm{v}1v=v

ベクトルは，ベクトル空間の元（げん，集合の要素） として定義します．
先に「ベクトル空間」が定義されるのが不思議ですね．
なお，性質1から4はベクトルどうしの演算について，5から8はベクトルとスカラーの演算についてを定義しています．
ベクトルどうしの掛け算や割り算は定義に含まれないことに注意しましょう．

ベクトル空間の定義から，4次元を超えるベクトルや複素ベクトルのように「大きさと向き」を定義しにくい量であっても，上記の8つの性質を満たすのであれば，何でもベクトルとみなすということになります．
ただし，ここまでイメージを広げてしまうと，後の議論が大変です．基本的には高校数学のベクトルのようなものを例として考え，具体的なイメージも捉えておくのが良いでしょう．

なお本稿では，上記のようにベクトルは太字で，スカラーは細字で表すこととします．

1.2. 基底

いくつかのベクトルa1,a2,⋯ ,an\bm{a}_1, \bm{a}_2, \cdots, \bm{a}_na1​,a2​,⋯,an​をそれぞれスカラー倍して足し合わせて作られたベクトル
c1a1+c2a2+⋯+cnanc_1\bm{a}_1 + c_2\bm{a}_2+ \cdots+ c_n\bm{a}_nc1​a1​+c2​a2​+⋯+cn​an​
を，線形結合といいます．

a1,a2,⋯ ,an\bm{a}_1, \bm{a}_2, \cdots, \bm{a}_na1​,a2​,⋯,an​の間に
c1a1+c2a2+⋯+cnan=0c_1\bm{a}_1 + c_2\bm{a}_2+ \cdots+ c_n\bm{a}_n =0c1​a1​+c2​a2​+⋯+cn​an​=0
という関係がc1=⋯=cn=0c_1=\cdots=c_n=0c1​=⋯=cn​=0以外で成立するのであれば，a1,a2,⋯ ,an\bm{a}_1, \bm{a}_2, \cdots, \bm{a}_na1​,a2​,⋯,an​は線形従属といい，そうでないとき，線形独立であるといいます．

あるベクトル空間VVVの元で，線形独立なe1,⋯ ,en\bm{e}_1,\cdots,\bm{e}_ne1​,⋯,en​があるとします．VVVの任意の元x\bm{x}xがe1,⋯ ,en\bm{e}_1,\cdots,\bm{e}_ne1​,⋯,en​の線形結合
x=x1e1+x2e2+⋯+xnen\bm{x} = x_1 \bm{e}_1 + x_2 \bm{e}_2 + \cdots + x_n \bm{e}_nx=x1​e1​+x2​e2​+⋯+xn​en​
として表せるとき，e1,⋯ ,en\bm{e}_1,\cdots,\bm{e}_ne1​,⋯,en​をVVVの基底と呼びます．さらに，VVVの基底がnnn個の元からなるとき，VVVはnnn次元であるといいます．

2. 写像

2.1. 写像の定義

ある要素aaaと，（別の）要素bbbが対応しているとき，以下のように表記します．
b=f(a)b = f(a)b=f(a)
ここで，fffをaaaからbbbへの写像と呼びます．
高校数学で習う関数と似たものに見えますが，写像は関数を拡張したものであり，aaaが連続に定義されていなくても構わないという特徴があります．例えば，fffが引数のジャンルを表す写像と定義して，
f(ねこ)=動物，f(おにぎり)=食べ物f(ねこ)=動物，f(おにぎり)=食べ物f(ねこ)=動物，f(おにぎり)=食べ物
とすることも可能です．

2.2. 線形写像

写像の中で，以下のような特別な関係を満たすものを線形写像といいます．

  1. f(a+b)=f(a)+f(b)f(\bm{a}+\bm{b}) = f(\bm{a})+f(\bm{b})f(a+b)=f(a)+f(b)
  2. f(ca)=cf(a)f(c\bm{a}) = c f(\bm{a})f(ca)=cf(a)

このような等号が一般の写像に対して成り立たないことは，例えば一般に
sin⁡(θ1+θ2)≠sin⁡θ1+sin⁡θ2\sin(\theta_1+\theta_2) \neq \sin\theta_1+\sin\theta_2sin(θ1​+θ2​)=sinθ1​+sinθ2​
であることからも理解できるでしょう．

任意の線形写像fffは，それに対応した行列AAAを用いて
f(x)=Axf(\bm{x}) = A\bm{x}f(x)=Ax
と書き表せることが証明できます．

3. 固有値と固有ベクトル

行列AAAで定められる線形写像は，2つのベクトルa\bm{a}aとb:=Aa\bm{b}:=A\bm{a}b:=Aaを特定のルールで関連付けています．
b:=Aa\bm{b}:=A\bm{a}b:=Aaという関係から，b\bm{b}bはa\bm{a}aを一定のルールで変換したものと捉えることもできます．

例えばa\bm{a}aとb\bm{b}bが2次元平面上のベクトルであるときを考えてみましょう．
行列AAAがよほど特殊な形をしていない限り，「b\bm{b}bはa\bm{a}aの大きさと向きを変えたもの」であり，それ以上はっきりとした関係を見出すのは難しいように見えます．

この行列AAAが正方行列であるとき，すなわち変換前と変換後のベクトルの次元が変化しないとき，変換をわかりやすく説明することができるのが，以下で紹介する，固有値と固有ベクトルです．

3.1. 定義

ある特別なベクトルv\bm{v}vに対して，以下のような関係がたまたま成り立っていたとしましょう．
Av=λvA\bm{v} = \lambda \bm{v}Av=λv
すなわち，ベクトルv\bm{v}vに対して行列AAAによる変換を施したとき，向きは一切変化せず，大きさだけがλ\lambdaλ倍に変化したということになります．
このようなAAAにとって特別なベクトルを，行列AAAの固有ベクトルといいます．また，倍率λ\lambdaλを行列AAAの固有値といいます．定義から，固有ベクトルと固有値は必ずセットで存在することになります．

固有ベクトルは，行列AAAの階数と同じだけ存在します．すなわち，AAAがn×nn\times nn×nの行列であったとき，（AAAがランク落ちしていない限り）nnn個の独立な固有ベクトルが存在することになります．

3.2. 線形写像を固有ベクトルで捉える

さて，一般にnnn次元のベクトルは，nnn個の独立なベクトルの線形和で表すことができることが示されています．この事実を用いると，任意のベクトルa\bm{a}aは，nnn個の固有ベクトルvi\bm{v}_ivi​と，それぞれに対応したスカラーcic_ici​を用いて
a=c1v1+c2v2+⋯+cnvn=∑i=1ncivi\bm{a} = c_1 \bm{v}_1 + c_2 \bm{v}_2 + \cdots +c_n \bm{v}_n = \sum^n_{i=1} c_i \bm{v}_ia=c1​v1​+c2​v2​+⋯+cn​vn​=i=1∑n​ci​vi​
と表すことができます．
これは，固有ベクトルvi\bm{v}_ivi​を基底としてベクトルa\bm{a}aを分解したと見ることもできます．

続いて，a\bm{a}aを行列AAAで変換したベクトルb\bm{b}bを考えてみましょう．このとき，上式を用いて
b=Aa=c1Av1+c2Av2+⋯+cnAvn=∑i=1nciAvi\bm{b} = A\bm{a} = c_1 A\bm{v}_1 + c_2 A\bm{v}_2 + \cdots +c_n A\bm{v}_n = \sum^n_{i=1} c_i A\bm{v}_ib=Aa=c1​Av1​+c2​Av2​+⋯+cn​Avn​=i=1∑n​ci​Avi​
となります．ここで，nnn個の固有値λi\lambda_iλi​を用いてAvi=λiviA\bm{v}_i = \lambda_i \bm{v}_iAvi​=λi​vi​と表せることを思い出すと，
b=Aa=c1λ1v1+c2λ2v2+⋯+cnλnvn=∑i=1nciλivi\bm{b} = A\bm{a} = c_1 \lambda_1\bm{v}_1 + c_2 \lambda_2\bm{v}_2 + \cdots +c_n \lambda_n\bm{v}_n = \sum^n_{i=1} c_i \lambda_i\bm{v}_ib=Aa=c1​λ1​v1​+c2​λ2​v2​+⋯+cn​λn​vn​=i=1∑n​ci​λi​vi​
となることがわかります．
すなわち，ベクトルa\bm{a}aの行列AAAによる変換は，「固有ベクトルviv_ivi​の方向に分割したa\bm{a}aの成分を，それぞれλi\lambda_iλi​倍すること」というふうに捉えなおすことができます．

まとめ：行列AAAによる変換は，固有ベクトルの方向ごとに捉えると簡単

4. 行列の対角化

すこしテクニカルな計算になりますが，ベクトルa\bm{a}aを新たな行列PPPとベクトルc\bm{c}cを用いて以下のように書き換えることができます．
a=[v1 v2 ⋯ vn][c1c2⋮cn]=Pc\bm{a} = \begin{bmatrix} \bm{v}_1\ \bm{v}_2 \ \cdots \ \bm{v}_n \end{bmatrix}\begin{bmatrix} c_1\\ c_2\\ \vdots \\ c_n \end{bmatrix} = P\bm{c}a=[v1​ v2​ ⋯ vn​​]​c1​c2​⋮cn​​​=Pc
ここで，正方行列PPPとベクトルc\bm{c}cを
P:=[v1 v2 ⋯ vn], c:=[c1c2⋮cn]P := \begin{bmatrix} \bm{v}_1\ \bm{v}_2 \ \cdots \ \bm{v}_n \end{bmatrix},\ \bm{c} := \begin{bmatrix} c_1\\ c_2\\ \vdots \\ c_n \end{bmatrix}P:=[v1​ v2​ ⋯ vn​​], c:=​c1​c2​⋮cn​​​
としています．同様の計算と定義によって，b=Aa\bm{b}=A\bm{a}b=Aaも
b=Pd\bm{b} = P\bm{d}b=Pd
と表すことができます．
このとき，
d=P−1APc=[λ10⋯00λ2⋯0⋮⋮⋱⋮00⋯λn]c\bm{d}=P^{-1}AP\bm{c} = \begin{bmatrix} \lambda_1 & 0 &\cdots &0\\ 0 & \lambda_2 & \cdots & 0\\ \vdots & \vdots & \ddots & \vdots\\ 0 & 0 & \cdots & \lambda_n \end{bmatrix}\bm{c}d=P−1APc=​λ1​0⋮0​0λ2​⋮0​⋯⋯⋱⋯​00⋮λn​​​c
となります．このような作業を，行列AAAの対角化と呼びます．
これはすなわち，
d1=λ1c1,d2=λ2c2,⋮dn=λncnd_1 = \lambda_1 c_1,\\ d_2 = \lambda_2 c_2,\\ \vdots\\ d_n = \lambda_n c_nd1​=λ1​c1​,d2​=λ2​c2​,⋮dn​=λn​cn​
ということです．カップリングしている項がなく，非常に簡潔な関係になっていることがわかります．
これは，適切な座標変換PPPによって，AAAによる変換はすべてデカップリングされたと読み取ることもできます．
すなわち，行列の対角化とは，行列による変換が最も簡潔に表せるような座標系（固有ベクトルを基底とする座標系）に取り直すことだと理解すると良いでしょう．

まとめ：行列AAAによる変換は，固有ベクトルを基底に取り直すと，対角行列で表せるようになる
