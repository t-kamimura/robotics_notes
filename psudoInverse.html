擬似逆行列

ムーア-ペンローズの擬似逆行列と，冗長な自由度を持つ連立方程式の解法についてまとめる．

行列A∈Rm×nA\in \mathbb{R}^{m\times n}A∈Rm×nが存在する．
AAAに対して，以下の性質を満たす行列XXXを考える．

  1. AXA=AAXA = AAXA=A
  2. XAX=XXAX = XXAX=X
  3. (AX)⊤=AX(AX)^\top = AX(AX)⊤=AX
  4. (XA)⊤=XA(XA)^\top = XA(XA)⊤=XA

このようなXXXのことをAAAの（ムーア-ペンローズの）擬似逆行列といい，A†A^\daggerA†（ダガー）あるいはA+A^+A+と表す．

線形連立方程式を解く

線形連立方程式
Ax=bA\boldsymbol{ x} = \boldsymbol{b}Ax=b
を解く．A∈Rm×nA \in \mathbb{R}^{m\times n}A∈Rm×n, x∈Rn\boldsymbol{x}\in \mathbb{R}^nx∈Rn, b∈Rm\boldsymbol{b}\in \mathbb{R}^mb∈Rmである．

正則な場合

m=nm=nm=nで，AAAが正則であればA−1A^{-1}A−1が存在するため，
x=A−1b\boldsymbol{x} = A^{-1}\boldsymbol{b}x=A−1b
が直ちに得られる．これは，線形連立方程式において未知数の数と式の数が一致している場合である．解は一意になる．

正則でない場合

もっと一般的な問題を考えよう．すなわち，未知数と式の数が一致しない場合，AAAは正則ではない．さらに場合分けをするなら，未知数に対して式が足りない場合(m<nm<nm<n)と，未知数に対して式が多すぎる場合(n<mn<mn<m)が存在する．

  • 前者は，拘束条件が少ないため，無数に解を持つことができる．
  • 後者は，拘束条件が多すぎるため，解が存在しない．

いずれの場合においても，解は∣∣Ax−b∣∣||A\boldsymbol{x}-\boldsymbol{b}||∣∣Ax−b∣∣を最小化するように探索する．前者では，原点から最も近い解を一つ選択することになる．後者では，最も近くまで近づけるものを解として扱うことになる．

拘束条件が少ない場合，ラグランジュの未定乗数法によって
x=A†b+(In−A†A)k\boldsymbol{x} = A^{\dagger}\boldsymbol{b} + (I_n-A^{\dagger}A)\boldsymbol{k}x=A†b+(In​−A†A)k
を得る．k∈Rn\boldsymbol{k}\in\mathbb{R}^nk∈Rnは任意のベクトルであり，解が無数に存在することを表す．

拘束条件が多すぎる場合，(二乗誤差の偏導関数)=0=0=0を解くことによって，
x∗=(A⊤A)−1A⊤b\boldsymbol{x}^* = (A^\top A)^{-1}A^\top \boldsymbol{b}x∗=(A⊤A)−1A⊤b
を得る．ただし，これは最適化の結果でしかなく，いかなる方程式の解でもないことに注意する必要がある．

例題

例えば冗長な自由度を持つロボットアームの逆運動学問題を考えてみよう．このとき，手先速度v\boldsymbol{v}vはヤコビ行列JJJと関節速度q˙\dot{\boldsymbol{q}}q˙​を用いて
v=Jq˙\boldsymbol{v} = J\dot{\boldsymbol{q}}v=Jq˙​
で与えられる．v∈R6\boldsymbol{v}\in \mathbb{R}^6v∈R6であるのに対し，冗長自由度アームではq˙∈Rn\dot{\boldsymbol{q}}\in \mathbb{R}^nq˙​∈Rn (n>6n>6n>6)である．このような場合，任意の手先速度は様々な関節角度の組み合わせによって得られる．すなわち，
q˙=J†v+(In−J†J)k\dot{\boldsymbol{q}} = J^\dagger\boldsymbol{v} + (I_n - J^\dagger J)\boldsymbol{k}q˙​=J†v+(In​−J†J)k

逆に，自由度に対して制御系が不足している劣駆動システムの場合，入力の制約から任意の出力を達成することは不可能である．
